O programa funciona passando os valores da arraylist como argumento para a função da classe operadora. É feita então uma iteração da lista em forma de array por cada elemento, somando o valor no mesmo como um double. No fim, o valor do total é retornado para o print final no main.

O teste feito em JUnit vai checar os valores enviados, e o valor esperado de retorno. Caso o teste dê errado, será acusado pelo jUnit.

Para rodar o programa, basta cloná-lo para uma IDE, montar e executar. Caso seja desejado o funcionamento em command prompt, basta navegar até a pasta 'src' com os arquivos .java, comentar a primeira linha de cada arquivo (package main;), rodar javac *.java, e depois java main.
