Exercicios para o processo seletivo para Estagiário de Desenvolvimento na Kopp Tecnologia, feitos por Robson Steindorff Ceolin.

Para o primeiro exercicio, foram utilizadas tecnologias de Java 11, na IDE Apache Netbeans. Para a implementação do teste unitário foi utilizado JUnit.

Para o segundo exercicio, foi criado um banco de dados utilizando PostgreSQL com auxilio de pgAdmin4. O banco de dados foi populado com unidades de teste aleatórias 
para poderem ser testadas as queries necessárias. Foi então enviado o modelo físico feito em draw.io.
